# initial-jekyll
Initial Repo for any New Jekyll Project with Browser-Sync

#Requirements
1. NodeJS
2. Gulp
3. Gulp-Shell
4. Lodash
5. Browser-Sync


# Steps for Setup
1. Clone Repo
2. Run the following commands
    1.  npm init
    2.  sudo npm install -g gulp
    3.  npm install --save-dev gulp-shell lodash gulp browser-sync
3. Run gulp
